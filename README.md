# AI+MCP Data Lineage

This project is an AI-powered data lineage system that integrates with MCP to discover and analyze scripts across servers.

## Architecture

- **Backend**: Python FastAPI
- **Frontend**: React + TypeScript
- **Metadata Store**: Gravitino (using PostgreSQL)
- **AI**: Ollama
- **MCP**: SSH integration
